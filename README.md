# Student Performance Analysis – Task 1 📊

## 📌 Overview

This project analyzes student performance data using Python to understand factors affecting academic results. It includes data exploration, basic analysis, and visualizations to extract meaningful insights.

---

## 🔍 Work Done

* Loaded and explored the dataset
* Checked data types, shape, and structure
* Verified missing values
* Removed duplicate records
* Performed basic statistical analysis
* Analyzed student performance trends
* Created visualizations for better understanding

---

## 📊 Key Analysis

* Calculated average final grade (G3)
* Counted students scoring above a threshold
* Found correlation between study time and performance
* Compared performance based on gender

---

## 📈 Visualizations

* Histogram of final grades
* Scatter plot (Study Time vs Final Grade)
* Bar chart (Average Grade by Gender)

---

## 💡 Key Insights

* Average student performance is moderate
* Very weak correlation between study time and final grades
* Male students have slightly higher average grades than female students (in this dataset)
* Most students score in the mid-range
